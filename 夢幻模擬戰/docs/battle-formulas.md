## 物理傷害

```
基礎傷害 = (攻擊者.攻擊 × (1 + 兵種攻擊加成%))
          - (防禦者.防禦 × (1 - 忽視防禦%) × (1 + 地形防禦加成% + 兵種防禦加成%))

若基礎傷害 < 0 → 歸零

最終傷害 = 基礎傷害
  × (1 + 技能威力%)          ← 技能倍率
  × (1 - 近戰懲罰%)          ← 遠程技能貼臉打才觸發
  × (1 + 物理傷害倍數/萬)      ← 攻擊者 buff
  × (1 + 物理受傷倍數/萬)      ← 防禦者 debuff
  × (1 + 暴擊傷害/萬)         ← 暴擊時才乘
  × (1 + 超級物理受傷倍數/萬)   ← 獨立疊算
  × (1 + 技能傷害倍數/萬)      ← 主動技能加成
  × (1 + 遠程受傷倍數/萬)      ← 遠程技能才觸發

最低保底 = 1
```

## 魔法傷害

結構與物理傷害相同，差異：
- 攻擊值改用 `智力`（但某些技能可用 `攻擊` 替代）
- 防禦值改用 `魔防`
- 倍數改用魔法系的 buff

## 治療

```
# 百分比治療
基礎治療 = 目標.最大生命 × (技能威力%)

# 魔法治療
基礎治療 = 施放者.智力 × (1 + 技能威力%)

# 最終治療
最終治療 = 基礎治療 × (1 + 施放者.治療倍數/萬 + 目標.受治療倍數/萬)
```

## 兵種克制系統

每個兵種組合有 4 個加成值：

| 參數 | 影響 |
|------|------|
| Attack | 攻擊者的攻擊加成% |
| Defend | 攻擊者的防禦加成% |
| Magic | 攻擊者的魔法加成% |
| MagicDefend | 攻擊者的魔防加成% |

> 具體的克制關係表（步兵→騎兵→飛兵→...）儲存在 ConfigData 中，2018 版未含實際數值。

### 克制判定顯示

```
優勢值 = 兵種攻擊加成 - 兵種防禦加成

若暴擊         → 顯示「暴擊」
若優勢值 > 0    → 顯示「克制」（Weak，傷害加強）
若優勢值 < 0    → 顯示「被克制」（Strong，傷害減弱）
否則           → 顯示「普通」
```

## 暴擊系統

### 暴擊率

```
# 英雄
暴擊率 = 技巧 × 10 + buff加成
上限 = 10000（100%）

# 士兵
暴擊率 = 設定值 + buff加成

# 實際判定
有效暴擊率 = 攻擊者.暴擊率 + 防禦者.受暴率加成
if random(0, 10000) < 有效暴擊率 → 暴擊
```

### 暴擊傷害

```
暴擊傷害倍數 = 基礎暴傷 + buff加成
最終傷害 *= (1 + 暴擊傷害倍數/萬)
```

## 地形加成

```
防禦者站在地形上：
  物理防禦 × (1 + 地形.BattleBonus%)
  魔法防禦 × (1 + 地形.BattleBonus%)
```

> 地形加成只影響防禦方。

## 屬性計算

### 英雄屬性

```
基礎 = (職業基礎值 + 職業成長 × 0.1 × (等級-1)) × (1 + 星級倍數/萬)
自身 = 基礎 × (1 + 自身倍數/萬) + 自身加值
成長 = 自身 × (1 + 成長倍數/萬) + 成長加值
最終 = 成長 × (1 + buff倍數/萬) + buff加值

下限 = 0
```

### 士兵屬性

```
基礎 = 士兵基礎值 + 士兵成長 × 0.1 × (英雄等級-1)
成長 = (基礎 × (1 + 成長倍數/萬) + 成長加值) × (1 + buff倍數/萬) + buff加值
指揮加成 = 成長 × 英雄指揮倍數/萬
最終 = 成長 + 指揮加成

下限 = 0
```

## 傷害計算流程

```
1. 基礎傷害 ← 攻擊 vs 防禦（含兵種克制、地形、忽視防禦）
2. × 技能威力
3. × 近戰懲罰（遠程貼臉時）
4. × 傷害類型倍數（物理/魔法 buff）
5. × 暴擊傷害（觸發時）
6. × 超級傷害倍數（獨立疊算）
7. × 技能傷害倍數（主動技能時）
8. × 遠程受傷倍數（遠程技能時）
9. 保底 1 點傷害
```
